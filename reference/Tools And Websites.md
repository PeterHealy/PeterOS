# Tools And Websites

## What This Page Is

A shared registry for concrete tools, websites, APIs, extensions, and products that come up in notes or source ingests. Keep entries short: URL, what the thing is for, and where it came from.

## Company Building

- [Clay](https://www.clay.com/) - GTM data, enrichment, intent, and AI workflow platform; useful for research, segmentation, enrichment, and outbound systems. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Graphed](https://www.graphed.com/) - AI marketing platform with agent, dashboard, and templated page workflows; relevant here because the notes reference Graphed-style ad and landing-page generation. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [HeyGen](https://www.heygen.com/) - AI video generation and localization tool; useful for scaling up winning creative into faster video production. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Keywords Everywhere](https://keywordseverywhere.com/) - browser-based keyword and SERP research tool; useful for turning search demand into landing-page and blog-post ideas. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Apollo](https://www.apollo.io/) - lead enrichment and sales engagement platform; useful for turning social engagers into contactable prospects. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Google Search Console](https://search.google.com/search-console) - search performance and indexing console; useful for sitemap submission, coverage monitoring, and page inspection. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Google Indexing API](https://developers.google.com/search/apis/indexing-api/v3/using-api) - API for notifying Google about certain page updates and removals; official docs say it is limited to job posting and livestream pages, so use it carefully. Mentioned in [[sources/company-building/marketing/Alex Lieberman - Claude Code Replaced My 20-Person Marketing Team]].
- [Substack](https://substack.com/) - publishing, email, subscription, and community platform; relevant as the infrastructure layer Rachitsky used to turn expert writing into a paid media business. Mentioned in [[sources/company-building/execution/Nate Martins - Reluctantly Influential]].
- [Lenny's Product Pass](https://www.lennysnewsletter.com/p/productpass) - bundle of software offers for Lenny's Newsletter paid subscribers; useful as an example of value-add bundling around a trusted audience. Mentioned in [[sources/company-building/execution/Nate Martins - Reluctantly Influential]].
- [Lenny's Data](https://www.lennysdata.com/) - AI-friendly Markdown archive of Lenny's Newsletter and podcast transcripts, plus related MCP/GitHub infrastructure; useful as a reference for making a content archive usable by LLMs. Mentioned in [[sources/company-building/execution/Nate Martins - Reluctantly Influential]].
