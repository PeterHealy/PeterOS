# Queries

Use this folder for durable answers that should not disappear into chat history.

Examples:

- comparisons
- decision memos
- research questions
- summaries created on demand

If an answer is likely to matter again, file it here and link it from the relevant evergreen, research, record, or reference pages.

If an answer is obviously better as an update to an existing canonical page, file it there instead of leaving it only in `queries/`.

The starter template lives at [[templates/saved-query]].
