# Inbox Sources

Use this folder as the first landing zone for pasted notes, copied excerpts, loose files, and rough source captures that have not been normalized yet.

Typical uses:

- a tweet you pasted from chat
- a podcast snippet you grabbed quickly
- a Kobo highlight batch
- a copied article excerpt
- a screenshot note with context

After ingestion, the material should usually be moved or rewritten into `raw/` and `sources/`.
