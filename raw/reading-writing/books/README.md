# Raw Book Sources

Use this folder for immutable book-related source captures.

Examples:

- Kobo highlight exports
- chapter note dumps
- clipped passages
- metadata stubs for books stored elsewhere

For Kobo or Obsidian imports, keep the full exact quote export here. The corresponding page under `sources/reading-writing/books/` should link back to this raw file, preserve exact personal notes, and synthesize the user's highlight pattern without duplicating the whole export.
