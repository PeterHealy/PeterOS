## User Notes Captured On 2026-04-27

- 1 person doing 20 people work from before.
- Growth consulting is very sought after. Pairs well with my GTM work at Seeley.
- GTM engineering started in 2023. Clay popularised it. Systems and processes rather than campaigns. Outbound, inbound, organic, paid ads. Distribution really.

### Process

- Get working on a new working directory with an agent.
- HTML to canvas to make a template for ad creation. Good for A/B testing.
- Use AI chat to find pain points for your ideal customer. What outcomes does the ideal customer want? Use tags like search Reddit so it is actual people.
- Get the output and put it into your agent.
- Brainstorm 40 ad titles and supporting paragraphs based on the source material.
- Then it creates all the ads.

### Summary So Far

- Deep research ICP fast.
- Use output from one LLM to feed into another LLM that creates header and ad text.
- Use personal software that turns that header and ad text into an image for the ad. That took 20 mins.
- Prompt example: "Build me a bulk ad generator. UI design claude skill."

### Iteration Loop

- Use results from Meta dashboard to see the best performer and then iterate or bulk create with that format.
- Use HeyGen for bulk creation, take the winner through vo3 to touch up. Maybe even hire a human at that point.
- Main purpose is fast iteration through the loop rather than class quality.

### Search And Landing Pages

- Tool: Keywords Everywhere after you Google search something.
- Take all keywords and copy them to agent to make a landing page for each keyword.
- After finding the winning formula, create landing page, blog posts, etc.
- Compare in a click campaign and do a $100 campaign over a 3 day period that gets the best performance.
- Then put the winners through a conversion campaign with their own dedicated budget.
- For the bulk creation of stuff there is a template page. See something like this: <https://www.graphed.com/page/test-page-integration>.
- Design and template is the same for all the different things; it is mainly the text on the page that changes.

### Indexing Notes

- Submit landing pages to Google Search Console so they get indexed.
- Ask the agent what sitemap these are under, hit the Search Console API to submit these so they get seen and the sitemap gets indexed by Search Console.
- If pages are not getting indexed, hit the web indexing API to do a manual indexing of that page.

### Bottleneck

- How do you understand what is actually happening?
- You can create thousands of versions of landing pages and ads, but you need to understand the data coming back so you can improve it.

### Positioning

- "You're not hiring me, you're hiring me AND my system."

### Another Workflow

- Fresh new window.
- Take engagers from a LinkedIn post.
- Enrich them and get email stuff via Apollo.
- Add to ad campaign.
- Use some email verifier to make sure emails are solid.
