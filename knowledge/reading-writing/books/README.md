# Books

Use this folder for living book pages and reading companions.

Recommended pattern:

- Keep one main page per book.
- Preserve the metadata style the user already uses in Obsidian where possible: `type`, `title`, `author`, `status`, `source`, `last_synced`, `tags`.
- Roll highlights, chapter notes, arguments, themes, and open questions into a single maintained page.
- Break out separate topic or entity pages only when they matter across multiple books or other sources.

The starter template lives at [[templates/book-page]].
