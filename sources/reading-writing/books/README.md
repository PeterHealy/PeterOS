# Book Source Pages

Use this folder for one canonical source page per book or imported book-highlight bundle.

Link each summary back to the raw book material under `raw/reading-writing/books/`.

Book pages based on Kobo or Obsidian highlights should be explicit that they are highlights-based. Do not summarize the full book unless the full text or a full reliable source summary was actually reviewed.

The point of this folder is not generic book reports. It is to make the user's extracted highlights useful:

- preserve a clickable link to the complete raw quote export
- preserve exact imported `My thoughts` notes on the source page
- group highlights into a `Highlight Map`
- synthesize only what the captured highlights support
- extract content ideas, build ideas, things to check out, and research leads only when they are concrete
- leave weak possibilities under `Possible Uses` instead of creating new pages

Default section pattern:

- `Summary Basis`
- `What This Page Is`
- `Highlight Map`
- `Your Captures`
- `Capture-Based Synthesis`
- `Reusable Ideas`
- `Possible Uses`
- `Open Questions`
- `Related Pages`
