Link each summary back to the raw article material under `raw/reading-writing/blogs-essays-articles/`.

For new URL-based sources, summarize the full article when accessible. For migrated Obsidian-highlight imports, mark whether the page is based on imported highlights only or has been upgraded to a full-source summary.
