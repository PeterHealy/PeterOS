# Templates

Reusable page skeletons for future ingests and maintenance work.

Use these only as starting points. Follow [[AGENTS]] and the local folder README over a template when there is a conflict.

## Available Templates

- [[book-page]] - highlight-led book or long reading-note page.
- [[evergreen-page]] - durable knowledge page.
- [[idea-capture]] - low-friction idea capture.
- [[record-blood-work]] - dated blood work record.
- [[reference-recipe]] - recipe reference page.
- [[research-dossier]] - active research file.
- [[saved-query]] - durable saved answer or memo.
- [[source-summary]] - canonical source hub.
- [[versioned-finance-plan]] - finance plan version.
